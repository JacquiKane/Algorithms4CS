Determine The Average Value In An Array

For an array of numbers, divide the sum of the numbers in the array, by the number of numbers in the array.

Let's use an example. Below, the array is called nums. The variable storing the average would be a double, as it may need to store a decimal value.


Sample Code - Assume Array Of Ints, nums


                int total = 0;
                for (int num : nums)
                    total += num;
                        
                double avg = total / nums.length;

How About Finding The Average Of Object Attributes?

The same algorithm applies to arrays containing objects. If you have an array of SoccerPlayer objects and you need the average age, then you would access the SoccerPlayer object's age attribute, add to a total, and then divide by the number of SoccerPlayer objects in the array.
