Determine Minimum Value In Array

Looking for the minimum value in an array can be done sequentially, by comparing values in an array. Assuming the array is not sorted, we can start with the element at index 0, and take that to be the assumed minimum. We traverse the array sequentially, checking for values that are less than the minimum. Of course, if the array is already sorted and we know it is, then the minimum value will either be at the start in element 0, if the array is sorted in ascending order, or the last element, if the array is sorted in descending order.

Let's use an example, with the first element assumed to be the minimum value, until we find another value as we traverse the array.


Sample Code, Assuming Array Of Ints - nums

                        
        int minVal = nums[0];
        for (int index = 1; index < nums.length; index++) {
            if (minVal > nums[index]) {
                minVal = nums[index];
                }
            }
